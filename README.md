# News Aggregator Full Stack Web App
Make sure to install the Beautiful Soup Python library, Flask, pyMySQL, and the python requests module.

Once the program is run, an HTTP web address will be provided in your console. Copy and paste that web address in your web browser.

This full stack web application enables users to access and view headlines from various news websites. The frontend of the application is designed using HTML, CSS and Bootstrap to make the website responsive. The Jinja2 templates are employed to display the data from the backend. The backend of the web application is developed using Python and the Flask web framework. The Beautiful Soup library is used to parse the HTML content of the news websites, and the Requests library is used to make HTTP requests to retrieve this content. The collected data is then stored in a MySQL database which is accessed using PyMySQL library.
